# i-gamehome-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。这里不针对任何特定网站或域名，仅作为个人笔记与页面的存放处。

## 项目简介

本仓库主要用于存放各类独立的 HTML 页面，内容涵盖学习笔记、工具页面、小游戏、实验性项目等。每个页面均为自包含的 HTML 文件，无需额外依赖即可在浏览器中直接打开。

## 目录说明

```
.
├── README.md          # 本文件
├── pages/             # 存放所有 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/            # 公共资源（如图片、样式、脚本等，可选）
```

- `pages/` 目录下存放所有归档的 HTML 页面，每个文件独立存在。
- `assets/` 目录用于存放页面可能共享的静态资源（若有），并非必需。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接通过浏览器打开查看。
- 页面内容可能包含个人笔记、工具、实验或娱乐性质的内容，无统一主题。
- 部分页面可能引用外部资源（如 CDN 上的库），但尽量保持自包含。
- 归档的页面不针对任何特定网站或服务，仅供存档与参考。
